Quais os grupos default?

Grupo Test - Tem acesso a todas as páginas de cadastro e algumas de controle
Grupo Admin - Tem acesso a tudo
Grupo Funcionário - Tem acesso aos Fluxos de compra e venda

Query para saber o acesso de cada grupo

	select descricao_grupo, descricao_papel from grupo 
	inner join grupo_papel on grupo.id_grupo = grupo_papel.id_grupo
	inner join papel on papel.id_papel = grupo_papel.id_papel;


Query para saber quantas páginas cada grupo pode ver
	
	select descricao_grupo, COUNT(descricao_papel) from grupo 
	inner join grupo_papel on grupo.id_grupo = grupo_papel.id_grupo
	inner join papel on papel.id_papel = grupo_papel.id_papel
	GROUP BY (descricao_grupo);
